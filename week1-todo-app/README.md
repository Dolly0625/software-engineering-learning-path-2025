# 第一週項目：CLI待辦清單應用

## 📋 項目概述

這是一個命令行待辦清單應用，展示了如何使用Python進行實際項目開發，並融入了資料結構的最優化設計。

## 🎯 功能

- ✅ **添加任務** - 帶優先級管理（高/中/低）
- ✅ **查看任務** - 列出所有任務
- ✅ **完成任務** - 標記任務為已完成
- ✅ **刪除任務** - 移除不需要的任務
- ✅ **任務統計** - 查看完成進度
- ✅ **優先級排序** - 按優先級查看任務
- ✅ **持久化存儲** - 任務保存到JSON文件
